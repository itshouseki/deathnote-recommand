What is Death Note?

Death note เป็นมังงะสัญชาติญี่ปุ่น แนวจิตวิทยา ถูกตีพิมพ์ขึ้นในปี 2003 ผู้เขียนเรื่องนี้มีทั้งหมด 2 คนด้วยกัน แต่งเนื้อเรื่องโดย อ.สึกุมิ โอบะ และวาดภาพโดย อ.ทาเคชิ โอบาตะ มังงะมีทั้งหมด 12 เล่ม ต่อมาในปี 2006 ก็ได้ถูกนำไปสร้างเป็นอนิเมะ โดยสตูดิโอ MADHOUSE มีทั้งหมด 37 ตอน อ้างอิงมาจากมังงะทั้ง 12 เล่ม และได้มีการดัดแปลงเป็นภาพยนตร์เข้าฉายในโรงภาพยนตร์ และมีนิยาย spin off ออกมา และในปี 2020 ก็ได้นำมาตีพิมพ์อีกครั้งในรูปแบบของ short stories ที่จะรวมเรื่องสั้นและ one shot ของตัวละครในเรื่องให้อยู่ ในเล่มเดียวจบ โดย Death Note เป็นเรื่องราวของ "ยางามิ ไลท์" นักเรียนมัธยมปลายอัจฉริยะที่บังเอิญพบสมุดโน้ตที่มีพลังพิเศษ ซึ่งถ้าหากเขียนชื่อของใครลงในสมุดคนนั้นจะต้องเสียชีวิต ไลท์ตัดสินใจใช้สมุดเล่มนี้เพื่อสร้างโลกในอุดมคติที่เขามองเห็นว่าไม่มีอาชญากรรม และตั้งตนเป็น "คิระ" เพื่อกำจัดคนชั่วร้าย แต่การกระทำของเขาก็ทำให้เขาต้องเผชิญหน้ากับ "แอล" นักสืบลึกลับที่ไม่เปิดเผยหน้าตาเข้ามาตามล่าเขา

Death Note : Manga vol.1 - 12

เรื่องย่อ : ไลท์ ยางามิ เด็กหนุ่มวัยมัธยมปลายได้พบกับสมุดเล่มหนึ่งหล่นลงมาจากฟ้าในขณะที่เขากำลังนั่งเรียนอยู่ หลังเลิกเรียนเขาได้เดินไปเก็บสมุดเล่มนั้น ซึ่งนั้นก็คือ Death Note เป็นสมุดที่เมื่อเขียนชื่อใครลงไปแล้ว คนๆนั้นจะต้องตาย

Death Note : Short Stories

เรื่องย่อ : เป็นหนังสือที่รวบรวมเรื่องสั้นที่เกิดขึ้นในจักรวาลของ Death Note ไม่ว่าจะเป็น เรื่องราวของผู้ใช้เดธโน้ตคนใหม่ อย่าง มิโนรุ เรื่องราวหลังจบคดีของคิระในเนื้อเรื่องหลัก ผู้ใช้เดธโน้ตคนแรก รวมไปถึง one shot ที่เล่าเรื่องราวของตัวละครในเนื้อเรื่องหลักอย่าง ไลท์ และ แอล ในมุมมองที่ไม่เคยมี ในเนื้อเรื่องหลัก เป็นต้น

Death Note Another Note : The Los Angelis BB Murder Cases

เรื่องย่อ : เรื่องราวเรื่องนี้เกิดขึ้นก่อนเหตุการณ์ที่ไลท์จะพบกับเดธโน้ต ตัวละครหลักจะเป็น มิโซระ นาโอมิ เป็นหนึ่งในอดีตเจ้าหน้าที่ FBI ฝีมือดีที่เป็นเหมือนกับลูกน้องของ L ในวันหนึ่ง ได้เกิดคดีฆาตกรรมต่อเนื่อง BB Case ที่ลอสแองเจลิส ทำให้นาโอมิที่ตอนนั้นได้พักงานจากการเป็นเจ้าหน้าที่ FBI รับคดีนี้ และได้เดินทางไปยังสถานที่เกิดเหตุ ซึ่งมันทำให้เธอได้พบกับ...ชายปริศนาที่มีทักษะและคงามเฉลียวฉลาด อีกทั้งท่าทาง ลักษณะคำพูด เหมือนกับ L มากๆ........

เรื่องย่อ : มีเนื้อหาดัดแปลงมาจากภาพยนตร์ไลฟ์แอคชั่น Death note : L change the world เป็นเรื่องราวหลังจากที่คดีคิระได้จบลงแล้ว ด้วยการที่แอลได้ทำข้อแลกเปลี่ยนที่ต้องแลกมากับสิ่งสำคัญที่เขาต้องเสียไป ในเวลาอีก 23 วันที่เขาเหลืออยู่ เขาได้ตัดสินใจลงมือสืบค้นและไขคดีสุดท้ายที่มีความท้าทาย การไขคดีครั้งสุดท้ายของ L จึงมีสิ่งต่างๆมากมายเกิดขึ้นและรอให้เขาได้เผชิญอยู่.....
